# Amplification-Roassal3

Test Amplification for Roassal3

## How to load
```smalltalk
Metacello new
  baseline: 'Amplification-Roassal3';
  repository: 'github://mabdi/Amplification-Roassal3/src';
  load.
```

TestAmplificationRoassel new amplifyAndVisualize
